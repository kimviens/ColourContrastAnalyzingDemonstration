Text Contrast Basics

Explanation:

To pass the minimum requirements for text contrast:

  • regular sized text should have a contrast of 4.5:1 against its background
  • large scale text (at least 18pt / 24px or if bold 14pt / 18.5px)

Exception:

  • decorative text

Non-Text Contrast Basics

Explanation:

To pass the minimum requirements for non-text contrast:

  • all parts necessary to understand the user interface component or graphical object must have a contrast ratio of at least 3:1 against adjacent colours

Exception:

  • decorative non-textual elements
